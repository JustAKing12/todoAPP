# TodoMVC App with React and Framer Motion

## Project functionalities:
* Add a task.
* Delete a task.
* Delete completed tasks.
* Edit an existing task.
* Filter tasks according to their status.

## Future features:
* Tasks by time.
* Saving data in the cloud or remote db.
* Priority of tasks.
